# React Native Calculator

A simple React Native calculator app that performs basic arithmetic operations.

## Installation 

- `git clone https://github.com/sphephelo44/calculator.git`
- `yarn install` (or `npm install` if you're using npm)

## Download and Install

1. Download the app using the provided link.
2. Install the APK file on your Android device.

---

This is my first React Native + Expo project. View the code and install the app to run and test it for yourself. Please note that this app is currently only available for Android devices.
